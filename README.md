# Shopee API Test Project

This project simulates a Shopee-like product subscription system using a credit-based API model. It demonstrates how APIs handle subscriptions, usage limits, and structured data flow.

---

## 🔍 Project Overview

This project replicates:

- Product subscription workflow
- Credit-based usage system (e.g., 20 credits)
- API request and response handling
- Structured data flow for automation

---

## ⚙️ Features

- Simulated subscription logic
- Credit tracking system
- REST API interaction
- Clean data handling
